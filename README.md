**Опис**

Репозиторій містить набір простих фронтенд-завдань. Кожне завдання реалізоване як окрема HTML-сторінка з відповідним JavaScript-файлом у папці `js` та спільним CSS у папці `css`.

**Структура проекту**

- [index.html](index.html) — головна сторінка проекту.
- [task-1.html](task-1.html) — Завдання 1 ([js/task-1.js](js/task-1.js)).
- [task-2.html](task-2.html) — Завдання 2 ([js/task-2.js](js/task-2.js)).
- [task-3.html](task-3.html) — Завдання 3 ([js/task-3.js](js/task-3.js)).
- [task-4.html](task-4.html) — Завдання 4 ([js/task-4.js](js/task-4.js)).
- [task-5.html](task-5.html) — Завдання 5 ([js/task-5.js](js/task-5.js)).
- [task-6.html](task-6.html) — Завдання 6 ([js/task-6.js](js/task-6.js)).
- [css/common.css](css/common.css) — спільні стилі.

**Призначення файлів**

- HTML-сторінки: кожна `task-N.html` демонструє окреме завдання та підключає відповідний `js/task-N.js`.
- `js/*.js`: логіка для кожного завдання.
- `css/common.css`: базові стилі.
